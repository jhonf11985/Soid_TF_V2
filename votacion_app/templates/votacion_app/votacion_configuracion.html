Cómo se calcula el quórum

{{ form.base_quorum }}

Presentes: usa sólo a los que están físicamente hoy.
{{ form.tipo_quorum }}

{{ form.valor_quorum }}
Si el tipo es Porcentaje, escribe un número entre 1 y 100 (por ejemplo: 50 = 50%).
Si el tipo es Cantidad, escribe cuántas personas como mínimo deben votar (por ejemplo: 80).
{{ form.votos_minimos_requeridos }}
Este valor se calcula automáticamente al guardar, combinando la base, el tipo y el valor de quórum. Indica cuántos votos mínimos se necesitan para que la elección sea válida.
Ejemplo rápido:
Si tienes 100 miembros con derecho a voto, eliges base Habilitados, tipo Porcentaje y valor 50, el sistema exigirá al menos 50 votos para validar la elección.
